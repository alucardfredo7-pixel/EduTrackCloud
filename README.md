# EduTrackCloud

EduTrackCloud is a cloud-based quiz web application developed using Flutter Web and Firebase services to support Sustainable Development Goals (SDG) learning activities.

## Technologies Used

* Flutter Web
* Dart
* Firebase Hosting
* Cloud Firestore
* Firebase Authentication

## Features

* User Registration and Login
* Quiz Categories
* Dashboard Monitoring
* Profile Management
* Public Read-Only Quiz Listing
* Cloud Deployment with HTTPS

## Deployment URL

https://edutrackcloud.web.app
